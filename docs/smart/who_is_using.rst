Feature Your Project
--------------------

To add your project to this list:

1. Fork the `documentation repository <https://github.com/JingtianYan/smart-docs>`_
2. Add your project details to this page
3. Submit a pull request

Or simply `open an issue <https://github.com/smart-mapf/smart/issues/new>`_
with your project information.

Required Information

* Project name and description
